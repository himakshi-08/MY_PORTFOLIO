# Himakshi Chadalavada Portfolio

Welcome to the source for **Himakshi Chadalavada's personal portfolio website**. This repository contains a responsive single-page portfolio built with modern web design, showcasing academic achievements, technical skills, internships, open-source contributions, projects, and contact information.

## 🚀 Project Overview

This portfolio is a static HTML/CSS website featuring:

- A clean landing page with a professional introduction
- About section highlighting education and internship experience
- Skills summary across frontend, backend, AI/ML, and engineering fundamentals
- Experience timeline including internships and open-source contributions
- Project showcase with interactive project cards and repository links
- Achievements and certifications with certificate links
- Contact section with email, LinkedIn, GitHub, and phone details
